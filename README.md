# TechStore CR - E-commerce Web

Aplicación web de comercio electrónico desarrollada como proyecto personal utilizando ASP.NET MVC, enfocada en la gestión de productos, usuarios y experiencia de compra.

---

## Tecnologías utilizadas

- ASP.NET MVC
- C#
- Entity Framework Core
- SQL Server
- HTML5
- CSS3
- Bootstrap
- JavaScript

---

## Funcionalidades

- Registro e inicio de sesión de usuarios (ASP.NET Identity)
- Panel administrativo para gestión de productos, categorías y subcategorías
- Carrito de compras funcional
- Persistencia de datos en base de datos SQL Server
- Navegación dinámica por categorías
- Diseño moderno y responsivo

## ⚙️ Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/TonTon63/TiendaTecnologica.git
```

2. Abrir el proyecto en Visual Studio
3. Configurar la cadena de conexión en appsettings.json
4. Ejecutar migraciones:
```
dotnet ef database update
```
5. Ejecutar el proyecto
